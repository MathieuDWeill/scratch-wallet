# 90-second demo script

## 0-10s — Problem

"Most autonomous trading bots ask users to connect valuable wallets. Scratch Wallet does the opposite: it starts with a tiny isolated wallet and hard limits."

## 10-25s — Setup

Show bankroll `100 USDC`, risk mode `Normal`, max trade size, max daily loss.

Say: "This is closer to a DeFi lottery ticket than a hedge fund, so the ticket must know when not to play."

## 25-45s — Scratch Today

Click **Scratch Today**.

Show:

- scanned opportunities
- most skipped
- one tiny play
- one fake claim blocked
- P&L path

## 45-65s — Explainability

Open Scratch Journal.

Say: "Every action has a reason: slippage killed the edge, liquidity was too thin, or Claim Shield blocked unsafe approval. Rejections are a feature."

## 65-80s — HashKey anchor

Show report hash and click **Real HashKey Anchor** if configured, otherwise **Mock Anchor**.

Say: "Decisions can be anchored on HashKey Chain so autonomous agent behavior becomes auditable."

## 80-90s — Close

"Scratch Wallet does not promise to beat the market. It makes autonomous small-wallet DeFi bounded, explainable, and safer by design."
