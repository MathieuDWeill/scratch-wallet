# product vision

Scratch Wallet is a risk-capped autonomous DeFi micro-wallet built by Mathieu D. WEILL for HashKey Chain.

Tagline: **A lottery ticket that knows when not to play.**

Core idea: users fund a tiny isolated wallet, choose a risk mode, and let an autonomous agent scan for small opportunities while hard bankroll rules, simulation, and Claim Shield keep the downside bounded.

Key message: **Autonomous finance should start with bounded downside.**
