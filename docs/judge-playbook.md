# Judge Playbook

The 90-second story:

1. Small-wallet DeFi feels like scratching a lottery ticket.
2. The dangerous part is not the tiny opportunity; it is connecting a valuable main wallet to unsafe approvals and uncontrolled bots.
3. Scratch Wallet isolates risk in a tiny wallet.
4. It scans HashKey Chain opportunities.
5. It rejects most candidates.
6. It blocks a fake claim.
7. It plays only a tiny opportunity that survives gas, slippage, liquidity, and mode limits.
8. It anchors the decision report on HashKey Chain.

Core line: **Autonomous finance should start with bounded downside.**
