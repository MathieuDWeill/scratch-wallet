# Launch thread draft

1/ I built Scratch Wallet for the HashKey Chain On-Chain Horizon Hackathon.

A lottery ticket that knows when not to play.

2/ Most autonomous trading bots ask users to connect a valuable wallet.

Scratch Wallet does the opposite: tiny isolated wallet, hard risk limits, simulation, and automatic stops.

3/ The agent scans for small DeFi opportunities: route edges, pool imbalances, incentives, and safe claims.

It rejects most candidates by design.

4/ The key idea: autonomous finance should start with bounded downside.

5/ Every played, skipped, rug-dodged, or stopped decision can be anchored on HashKey Chain through ScratchWalletRegistry.
