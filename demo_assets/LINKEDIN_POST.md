I built Scratch Wallet for the HashKey Chain On-Chain Horizon Hackathon.

Scratch Wallet is a risk-capped autonomous DeFi micro-wallet. Instead of connecting a main wallet to an autonomous trading bot, users fund a tiny isolated wallet, choose a risk mode, and let the agent scan for small opportunities.

The agent simulates every action, blocks unsafe approvals, rejects most opportunities, and stops automatically when loss limits are reached.

The product thesis is simple: autonomous finance should start with bounded downside.

Tagline: A lottery ticket that knows when not to play.
