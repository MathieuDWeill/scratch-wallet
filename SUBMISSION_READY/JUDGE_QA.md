# Scratch Wallet — Judge Q&A

## Is this a trading bot?
It is a risk-capped autonomous micro-wallet, not a price prediction bot. The agent only operates inside a tiny isolated wallet and rejects most opportunities.

## What is the core innovation?
The product design starts from bounded downside. Users do not connect their main wallet. Every action is simulated, unsafe claims are blocked, and every decision can be anchored on HashKey Chain.

## Why HashKey Chain?
HashKey Chain acts as the audit layer for autonomous agent behavior: played opportunities, skipped opportunities, rug dodges, and stop events can be recorded transparently.

## Does it use real money in the demo?
The demo is safe by default and runs in dry-run/mock mode. A real HashKey anchor path is included for a deployed registry contract using a tiny burner wallet.

## How could it make money?
Potential monetization: premium hosted bots, strategy modules, Telegram alerts, performance fee on realized gains, and protocol-specific tiny-wallet agents for liquidity testing.

## What is the memorable line?
A lottery ticket that knows when not to play.
